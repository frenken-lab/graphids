Training Logic & Architecture

GATv2 Upgrade

  • Switched all configs from conv_type: "gat" to conv_type: "gatv2"
  • Key difference: GATv2 uses edge features (edge_dim=11) in message passing
  • Standard GAT ignores edge attributes entirely
  • Added proj_dim config field for optional pre-attention projection

Brody et al. 2022 — “How Attentive are Graph Attention Networks?”

DQN Fusion Vectorization

Rewrote DQN training from per-sample Python loops to vectorized PyTorch ops:

  • ~55x speedup — eliminated ~10M Python iterations per run
  • Replay buffer, Q-values, and target updates all operate on batched tensors
  • dqn.py: 110 → 772 LOC (cleaner separation of concerns)

Model Construction Fix

  • All sites now use Model.from_config(cfg, ...) — never manual Model(param1=...)
  • Prevents silent divergence when config fields are added
  • VGAE decoder: added feature normalization for stable reconstruction loss
